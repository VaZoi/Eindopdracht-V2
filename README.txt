### Installation Instructions:

1. Downloaded the Zip folder and unpack it.

2. Move the unpacked folder to the C:\xampp\htdocs location

3. Start xammp and start Apache and MySQL.

4. Go to http://localhost/phpmyadmin/ and import the supplied .sql file in a database that is named vzrent

5. Go to http://127.0.0.1/Eindopdracht%20V2/

6. Login with the supplied Admin Account

7. done
